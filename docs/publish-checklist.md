# Publish Checklist

Before making the repository public, try to finish these steps:

## Must Have

- confirm the repo name is `ai-pe-lab`
- upload the current directory to GitHub
- enable GitHub Pages from the default branch
- check that `examples/three-stage-adventure/` opens publicly
- add the repo description and topics from `docs/github-copy.md`

## Strongly Recommended

- add 4 to 6 screenshots to the root README
- record a 20 to 40 second demo video
- create the first tagged release
- open at least 2 issues yourself to show roadmap and maintenance direction

## Nice To Have

- add a second example under `examples/`
- write a short bilingual project post
- collect one or two real teacher or student usage notes
